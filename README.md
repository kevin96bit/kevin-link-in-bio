# Link-in-Bio con Tailwind CSS

Questa è una versione moderna del progetto **Linktree-like**, 
costruita con **TailwindCSS via CDN**.

# Update

Ogni volta che farò il push da GitHub, Vercel rileverà la modifica e aggiornerà il sito 
tutto ciò che dovro fare è : 

git add .
git commit -m "descrizione breve della modifica"
git push

 **Deploy su Vercel:**
   - Vado su vercel.com → *Add New Project* → *Import from Git*.
   - Oppure devo usare il terminale:
     ```bash
     npm i -g vercel
     vercel
     ```
